绑定域名

现在我们已经可以无碍的使用 Hexo 来写博客并部署到GitHub上了，可是有一个地方不太优雅，那就是域名。因为我们使用的是 Github 提供的二级域名 github.io，一来太长，二来也不太够自定义、个性化。

所以，我们可以去买一个域名，然后解析到 Github 去，国内的万网、西部数码，国外的 Godady 等网站都可以购买。我的域名 .tech 每年才 5 元，价格实惠。实在不济，还有 .tk，.ml，.cf，.ga 四个免费域名提供开放注册（为互联网的免费精神干杯🍻）。

购买域名之后，我们只需要在购买域名的网站把 DNS 解析创建一个CNAME记录和两个 A 记录到：
